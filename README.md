# ChatGPT Bulk Unarchive Extension

A Chrome extension that automatically unarchives all archived chats in ChatGPT.

## Features

- Bulk unarchive archived chats
- Start / Stop automation controls
- Live progress counter
- Works inside Archived Chats panel

## Installation

1. Download this repository
2. Open Chrome → chrome://extensions
3. Enable Developer Mode
4. Click Load unpacked
5. Select this folder

## Usage

1. Open ChatGPT
2. Go to Settings → Data Controls → Archived Chats
3. Click extension icon
4. Press Start
